Nokia’s Quality Assurance Solution’s Execution

The Quality Assurance solution’s frontend was developed using ReactJS and Python, while its backend was developed using NodeJS. Furthermore, we used OpenCV, TensorFlow, CNN, and IoT to develop the advanced solution and integrate it with Nokia’s existing security cameras at the manual packaging station.
